记一次 Hexo 博客搭建全过程

3月 6, 2019

Hexo 作为 众多静态博客框架 之一，具有快速、简洁且高效的特点，可基于 Node 环境快速搭建博客，并且拥有众多美观的 主题 可以选择。

今天，将详细记录一次博客搭建的全过程，主要包括 博客的初始化、主题的更换、部署到 Github Pages、Hexo Admin 的集成、 TravisCI 持续集成、博客在线编辑 等。
